Manipulation Framework - Basic Interaction

Press Ctrl+M to toggle Manipulation Mode.

  • Drag: Move element.
  • Alt + Drag: Resize element.
  • Shift + Drag: Duplicate element.
  • Ctrl + Click: Multi-select elements.
  • F1 / F2 / F3: Switch Layer (Document / Template / Page).

Note: In Manipulation Mode, only elements in the active layer are visible.
